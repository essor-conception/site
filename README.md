Site vitrine – Prévention Incendie, Formation & Conseil (Région Occitanie).

## Déploiement sur GitHub Pages

```bash
npm install
npm run deploy
```

Puis dans le dépôt GitHub : **Settings → Pages** → Source **Deploy from a branch** → branche **gh-pages** → dossier **/ (root)**.
